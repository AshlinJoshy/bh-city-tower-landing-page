# Replacing the placeholder images

Every image in this page is a placeholder SVG generated by
`scripts/make-placeholders.mjs`. They live in `public/images/` and use
gradient fills that loosely match the Figma palette so the page looks
intentional even before the real assets are in.

## How to swap in the real Figma assets

1. Open the Figma file → `Final` page → `CIty Tower landing page op3-1`.
2. Click a layer in the canvas (e.g. one of the dining-area photos).
3. In the right sidebar → **Export** → choose `PNG` or `JPG` → click
   `Export <layer name>`.
4. Save the exported file into `public/images/` using **exactly** the
   filename listed below.

Existing components reference the images by filename, so any matching
file (`.svg`, `.png`, `.jpg`, `.webp`) will be picked up automatically
as long as you keep the same base name.
